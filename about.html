About This BJC Tool

BJC Bandlocking Tool is a mobile-friendly web app designed to make it easy to:

  • Log in to your ZLT S10G router (Admin/User)
  • Lock onto specific LTE bands (Band 3, 28, 40, 41)
  • Generate valid IMEI numbers for device testing
  • Search online using a built-in Google search

This tool works offline (for IMEI and UI), and can be packaged into an APK.

🔒 Please use responsibly. This tool is for educational and diagnostic purposes only.
